## Installing

1. Clone down the repo:

` git clone git@github.com:tmaria17/sweater-weather.git`

2. Run:

`bundle install`

`bundle update`

3. Setup the database:

 `rake db:setup`



## Running the tests
To run the full testing suite, run `rspec` from the command line.
